# Sea Level Predictor

FreeCodeCamp Data Analysis with Python - Project

## Project Description
Analysis and prediction of global sea level rise using historical data from 1880-2013. This project uses linear regression to forecast sea level changes up to 2050.

## Features
- **Data Visualization**: Scatter plot of historical sea level data
- **Linear Regression**: Two lines of best fit:
  1. Using all data (1880-2013)
  2. Using recent data (2000-2013)
- **Future Prediction**: Extrapolation to year 2050
- **Comparative Analysis**: Compare long-term vs recent trends

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scipy
- NumPy
